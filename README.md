# Pixmenu

Pixmenu is a game utility developed in C++. It functions without the need for memory reading, making it compatible with a wide range of games. By adjusting the configuration file, it can be adapted to work with any FPS game that features enemy highlight colors, such as VALORANT or Overwatch.

Pixmenu assists with aiming and shooting by generating mouse input. It is recommended that users replace the mouse movement functionality with their own implementation.

## Function Overview

### Aim assist

- Identifies targets by scanning the screen for pixels within a specified color range.
- Automatically moves the aim towards the detected target.

### Triggerbot

- Automatically fires when the crosshair is positioned over an enemy.

### Rapid-fire

- Continuously shoots while the crosshair remains on a target.

### Recoil Control

- Automatically compensate the player's crosshair recoil.

### Prediction Aim

- Predicts the target's movements and aims in advance.
